<h1>jogo do amigo secreto</h1>

<h2>Sobre</h2>
<p> Nesse desafio proposto pela Alura, o intuito era testar minha lógica de programação e a forma como eu poderia resolver o problema. O desafio consistia em desenvolver um algoritmo que permitisse adicionar participantes e realizar o sorteio de um amigo secreto entre eles.</p>
